# Calculator Application

This is a simple command-line calculator application that evaluates mathematical expressions.

## How to Run

To run the calculator, execute the `main.py` file with your desired expression as an argument:

```bash
python main.py "3 + 5"
```

The application will output the result in a JSON format.

## Examples

- Addition:
  ```bash
  python main.py "10 + 2"
  ```

- Subtraction:
  ```bash
  python main.py "10 - 2"
  ```

- Multiplication:
  ```bash
  python main.py "10 * 2"
  ```

- Division:
  ```bash
  python main.py "10 / 2"
  ```

- Complex Expression:
  ```bash
  python main.py "2 * 3 - 8 / 2 + 5"
  ```

## Running Tests

To run the unit tests for the calculator, execute the `tests.py` file:

```bash
python tests.py
```
